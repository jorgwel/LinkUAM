Descripción del contenido de los directorios:

/baseDeDatos
============
Contiene una documentación extensa de la base de datos y el script de inicialización de la base de datos de LinkUAM

/codigoFuente
=============
Incluye el código fuente de LinkUAM

/manualDesarrollador
====================
Incluye documentación sobre las apis y el código generado durante el desarrollo de LinkUAM

/programas
==========
Contiene todo el software necesario para montar LinkUAM en un servidor de aplicaciones y/o en un entorno de desarrollo en caso de no contar con una conexión a interten.

/reporte
========
Incluye el reporte del proyecto terminal, así como algunos diagramas adjuntos que no eran visibles a detalle dentro del documento.

/war
====
Incluye los archivos necesarios para solamente desplegar la aplicación, no para continuar su desarrollo.

GuiaDeInstalacion.pdf
=====================
Contiene instrucciones para instalar LinkUAM y/o continuar su desarrollo.
